# **App Name**: PathMind Pro

## Core Features:

- Assessment Introduction & Start: Displays a centered card layout with assessment title, subtitle, optional illustration, progress indicator, and a call-to-action button to begin the assessment.
- Dynamic Question Presentation: Manages the display of assessment questions one at a time, including standard questions with radio button options and branching scenario questions with decision cards, with smooth transitions and navigation controls.
- Assessment Score & Performance Feedback: Presents the user's assessment score, performance messages (e.g., 'Excellent'), and a call-to-action to proceed to the action plan, featuring success animations or indicators.
- Action Plan Commitment Form: Provides a card-based input form for users to define up to three commitment fields, add more commitments dynamically, select a timeline (e.g., 1 week, 1 month), and set priorities.
- Plan Submission Confirmation: Displays a success screen with a checkmark icon, confirmation message that the plan has been saved, and subtext about reminders, culminating in a 'Finish' button.
- AI Follow-up Nudge Tool: A generative AI tool that crafts personalized, friendly, and relevant follow-up nudge messages based on the user's action plan commitments for different predefined timelines (e.g., Day 1, Day 3, Day 7).
- Smart Reminders Preview: Showcases the AI-generated follow-up nudges as notification-style cards, each with an icon, message preview, and blue/green highlights, indicating future automated communication.

## Style Guidelines:

- Primary action color: A vibrant professional green (#22C55E) symbolizing growth and success, suitable for calls to action and active states, in keeping with the requested 'Green' theme.
- Secondary accent color: A deep, complementary blue (#2563EB) for highlights, selected elements, and gradients, adhering to the requested 'Blue' theme and creating a dynamic contrast with the primary green.
- Background color: A very light, desaturated greenish hue (#F2F9F4) providing a clean, subtle base that supports readability and aligns with the overall light theme and primary color palette.
- Body and headline font: 'Inter', a modern grotesque sans-serif that offers excellent readability and a neutral, professional aesthetic suitable for an enterprise-grade LMS module.
- Use minimal, clear, and professional icons that support navigation and content understanding without distracting from the core experience.
- Employ a consistent, minimal, and distraction-free card-based layout with uniform padding (desktop-first for responsiveness), soft shadows, and rounded corners (12px–16px radius) for a polished, modern feel.
- Incorporate smooth fade and slide transitions between screens, subtle hover effects on interactive elements, loading skeleton UIs for a sense of realism during content loading, and micro-interactions for an enhanced user experience.
